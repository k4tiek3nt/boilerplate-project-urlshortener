URL Shortener Microservice

URL Shortener

Example Usage:

1. Type in a valid URL. (i.e. https://www.freecodecamp.org/)

2. Click "POST URL".

Note: This will redirect to a JSON page that looks like: {"original_url":"https://www.freecodecamp.org","short_url":2}

Example of accessing shortened URL:

If you click this link: https://boilerplate-project-urlshortener.k4tiek3nt.repl.co/api/shorturl/2 it will redirect to https://www.freecodecamp.org/

Note: Your shortened URL will be a different number than 2. You can access your new URL by typing in the address bar "https://boilerplate-project-urlshortener.k4tiek3nt.repl.co/api/shorturl/+[Your Short URL #]"
